BEFORE RUNNING:
Input items you want to auction in input.txt file

Format is
item_name starting_bid min_bid duration

File can only handle 4 items, inputting more will crash program

1. Start server on localhost

- Run "Main.java" in ServerSide project

2. Open"javafxloginvideo" project
- Run "HelloApplication.java"

3. Login/Register on home page

4. Set bid at top

5. Press buttons to send bids to correlated items
